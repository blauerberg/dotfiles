# dotfiles

My dotfile is optimized for Archlinux, i3wm, zsh(based by [prezto](https://github.com/sorin-ionescu/prezto)) and Full HD display.

# Getting Started

```
git clone https://github.com/blauerberg/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./bootstrap.sh
```

# Prerequisite

- git
- sudo
- zsh
